# Example: IVI Assistant

An in-vehicle assistant helps users with infotainment and navigation-related tasks.

## Accountability design
- Product lead accountable for release decision
- Engineering lead accountable for technical readiness
- Operations owner accountable for post-release monitoring
- Named owner for deterministic fallback path

## Key triggers
- rising low-confidence rate
- unsupported-command concentration
- repeated fallback activation
- unsafe ambiguity around user intent
